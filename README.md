## Gestire?

Sim, um nome fácil, que fica no ouvido e tem tudo a ver com aquilo que o projeto se compromete. Gestire, uma palavra italiana, significa gerir. Simples assim.

## Que raio é isto?

Com este projeto, pretende-se implementar uma plataforma capaz de gerir a disponibilidade de salas e equipamentos para uma determinada entidade (neste caso o nosso querido DETI).

## Então isto vai ser o quê?

Um programa de terminal não é de certeza. Ainda temos de discutir se vamos fazer uma webapp (é o mais certo) ou aplicações nativas (dá muitíssimo trabalho).
